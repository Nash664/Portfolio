Project Sample 3: Intelligent Gomoku Game

Java AI Algorithms Game Theory

Overview: A strategic board game implementation of Gomoku (Five in a Row) featuring both player vs player and player vs AI modes with multiple difficulty levels.

My Role: Sole developer responsible for game mechanics, AI implementation, algorithm optimization, and user interface design.

Challenge: Developing an AI opponent that could make intelligent moves within reasonable time constraints while providing an appropriate level of challenge for different skill levels.

Solution: Implemented the minimax algorithm with alpha-beta pruning to efficiently search the game tree, added depth-limited search and position evaluation heuristics for varying difficulty levels, and optimized board state representation for performance.

Results: A fully functional Gomoku game with an engaging UI, multiplayer support, and a challenging AI opponent capable of playing at different skill levels with efficient move calculation.

Key Skills Applied: Algorithm implementation, artificial intelligence, adversarial search, performance optimization, Java Swing UI development
